# CoTS Bifurcation Project

Simulation and bifurcation modelling of crown-of-thorns starfish (CoTS) outbreak dynamics on Koh Tao reefs.

This project adapts bifurcation theory for local CoTS density data, with an emphasis on testing whether published threshold values are appropriate for Koh Tao. The workflow includes data cleaning, exploratory summaries, parameter testing, bifurcation modelling, simulation modelling, and sensitivity analysis.

## Repository Structure

| File / Folder            | Description                                                                                            |
| ------------------------ | ------------------------------------------------------------------------------------------------------ |
| `00_RUN.R`               | Master script that runs the full workflow in sequence.                                                 |
| `01_CLEAN.R`             | Loads and tidies raw CoTS, substrate, and site-level data.                                             |
| `01.1_EXPLORE.R`         | Generates basic summaries, density checks, and exploratory plots.                                      |
| `02_PARAMETERS.R`        | Defines model parameters, threshold values, and alternative parameter scenarios.                       |
| `03_BIFURCATION_MODEL.R` | Runs the core bifurcation model using selected parameter values.                                       |
| `04_SIMULATION_MODEL.R`  | Runs simulation scenarios to explore CoTS dynamics over time.                                          |
| `05_SENSITIVITY.R`       | Tests sensitivity of outcomes to alternative thresholds, initial densities, and parameter assumptions. |
| `06_PLOTS.R`             | Produces final figures and summary tables.                                                             |
| `R/`                     | Helper functions for cleaning, modelling, simulation, and plotting.                                    |
| `data/raw/`              | Raw input data. Not tracked on GitHub.                                                                 |
| `data/processed/`        | Cleaned datasets generated by the workflow.                                                            |
| `outputs/`               | Model outputs, figures, tables, and saved objects. Not tracked on GitHub.                              |
| `docs/`                  | Notes on project rationale, assumptions, and modelling decisions.                                      |

## Analysis Summary

### Data Cleaning

* Reads raw CoTS and site-level data.
* Standardises site names, dates, density calculations, and substrate variables.
* Converts CoTS counts to density per hectare where required.

### Exploration

* Summarises observed CoTS densities across sites.
* Compares observed densities against published outbreak and Allee threshold values.
* Identifies site-level maxima, means, and medians for parameter testing.

### Parameter Setup

* Defines alternative threshold scenarios.
* Compares published values with Koh Tao-specific observed density ranges.
* Allows sensitivity testing across plausible endemic, outbreak, and collapse thresholds.

### Bifurcation Model

* Implements the deterministic bifurcation model.
* Tests how equilibrium outcomes change under alternative threshold assumptions.

### Simulation Model

* Runs time-based simulations of CoTS density trajectories.
* Allows initial densities, growth rates, mortality terms, and threshold values to vary.
* Intended as a proof-of-concept model rather than a formal predictive model.

### Sensitivity Analysis

* Tests whether model conclusions are robust to parameter uncertainty.
* Focuses especially on the placement of critical thresholds relative to observed Koh Tao densities.
